# 목표: 버튼을 누르면 운세를 무작위로 뽑아서 알려주는 웹
 - 
  ## 규칙 
   - "오늘의 운세는 한번만!" 이라는 문구를 넣는다.
    - 나쁜 운세가 나오면 "이런점을 조심하면 행운이 나와요!" 같은 조언형태의 메세지를 준다.
 
 ## 디자인 
  - 레트로 픽셀 아트(Y2K) 감성.귀여운 픽셀 그래칙, 레트로한 폰트를 사용해 문방구앞 뽑기 기계를 돌리는듯한 친근, 캐주얼한 느낌을 준다.

  ## 기능
  - 운세결과 하단에 행운의 컬러,음식,행동 중에서 무작위로 하나를 운세에 넣는다.
  - 사용자가 화면의 카드를 클릭, 셔플할떄 뒤집히는 생동감을 주는 에니메이션 효과를 넣는다. 
  - 운세는 3줄에서 4줄정도 적당량의 길이로 작성한다.
