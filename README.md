# 두근두근 오늘의 운세 뽑기

문방구 앞 뽑기 기계를 돌리는 듯한 느낌으로, 클릭 한 번에 오늘의 운세를 무작위로 뽑아보는 레트로 픽셀(Y2K) 감성의 웹 게임입니다.

## 게임 개요

- **컨셉**: 서버/로그인 없이 누구나 바로 즐기는 한 페이지짜리 운세 뽑기 게임
- **디자인**: 레트로 픽셀 아트(Y2K) 감성 — 각진 픽셀 테두리, 하드 섀도우, 도트 폰트(Galmuri), 파스텔+비비드 컬러 조합의 문방구 뽑기 기계 룩
- **콘텐츠**: 좋음/보통/나쁨 3가지 톤의 운세 20종, 행운의 컬러·음식·행동 각 10종

## 게임 방법

1. 화면 중앙의 뽑기 카드(물음표 캡슐)를 클릭합니다.
2. 카드가 셔플되며 흔들리다가 톡- 하고 뒤집히며 오늘의 운세가 공개됩니다.
3. 운세는 3~4줄로 표시되며, 하단에 **행운의 컬러 🎨 / 음식 🍬 / 행동 ✅** 중 무작위로 하나가 뱃지로 함께 나옵니다.
4. 나쁜 운세가 나온 경우, 마지막 줄에 💡 조언 문구("~만 조심하면 행운이 나와요!")가 강조되어 표시됩니다.
5. "오늘의 운세는 한번만!" 문구는 뽑기 전 재미로 보여주는 문구이며, 실제로는 **"한 번 더 뽑아보기"** 버튼으로 몇 번이든 다시 뽑을 수 있습니다.

## 게임 실행 방법

이 프로젝트는 별도 빌드 과정이 없는 정적 웹앱(HTML/CSS/JS)입니다. 단, `js/main.js`가 `content/fortunes.json`을 `fetch`로 불러오기 때문에 `index.html`을 더블클릭해 `file://`로 직접 열면 브라우저 보안 정책(CORS)에 막혀 동작하지 않습니다. **반드시 로컬 서버로 실행**해 주세요.

### 방법 1 — Node.js (`serve`)

```bash
cd D:\workspace\pick
npx serve .
```

터미널에 출력되는 주소(예: `http://localhost:3000`)로 브라우저에서 접속합니다.

### 방법 2 — Python

```bash
cd D:\workspace\pick
python -m http.server 5500
```

브라우저에서 `http://localhost:5500` 으로 접속합니다.

### 방법 3 — VS Code

VS Code의 "Live Server" 확장을 설치한 뒤 `index.html`에서 우클릭 → **Open with Live Server**를 선택합니다.
